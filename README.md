# Fitting of QYs

This repository contains the raw data and the fitting routines in for the determination of the reaction quantum yields of photoisomerization. See [JOC publication](... TBA) for reference. To successfully run the Jupyter Notebook `QY fitting.ipynb` at least Python 3.7 and the following packages are required:

* [NumPy](https://numpy.org/)
* [Scipy](https://scipy.org/)
* [LMFIT](https://lmfit.github.io/lmfit-py/)
* [Numba](https://numba.pydata.org/)
* [Matplotlib](https://matplotlib.org/)
* [Pandas](https://pandas.pydata.org/)
